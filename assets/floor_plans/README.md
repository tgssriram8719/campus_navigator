# Floor Plan Assets

Place your floor plan images in this folder.

## Recommended Format
- **File type**: PNG (recommended) or JPEG
- **Size**: 1000-2000 pixels wide
- **Naming**: `ground_floor.png`, `first_floor.png`, etc.

## Example
```
floor_plans/
├── ground_floor.png
├── first_floor.png
└── second_floor.png
```
